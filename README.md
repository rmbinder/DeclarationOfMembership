# DeclarationOfMembership

'DeclarationOfMembership' is an Admidio plugin. It creates an online membership declaration.

### Installation and Update

1. Create a folder named 'DeclarationOfMembership' under adm_plugins
2. Copy all plugin files into this folder (If the folder already exists, you only need to replace all the files within it.)
3. To install (or update), run the following PHP file: .../adm_plugins/DeclarationOfMembership/system/install.php
